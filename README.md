# MirrorIS_Server
MirrorIS Server

### Установка:
	1. pip3 install -r requirements.txt
	2. python3 ./commands.py
	3. python3 /app.py

### Запросы:
	/repository
	/repository/create
	/repository/1
	/repository/1/update
	/repository/1/delete
	/repository/1/run
	/repository/1/reset

	/user
	/user/create
	/user/1
	/user/1/update
	/user/1/delete

	/task
	/task/1

### Методы:
	PUT -> update
	POST -> create
	DELETE -> delete
	GET -> все остальные
